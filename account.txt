Windowsのスタートメニュー（画面下の田マーク）をクリックし、「資格情報」 と入力します。

検索結果に出てくる 「資格情報マネージャー」 をクリックして開きます。

「Windows 資格情報」 というボタン（またはタブ）をクリックします。

一覧の中から git:https://github.com という項目を探します。

その項目の右側にある矢印（∨）をクリックして詳細を開き、「削除」 をクリックします。

確認画面が出るので「はい」を押します。

これで、パソコンから「親御さんのログイン情報」が消えました。

手順 2: このプロジェクトの署名を元に戻す
先ほど、このゲームのフォルダ（ultimate-game）での作業者名を親御さんの名前に変更しました（git config --local）。 もし、今後お子様がこのフォルダで作業をする際に、お子様の名前で記録を残したい場合は、以下のコマンドで元に戻します。

VSCodeなどのターミナルで以下を実行してください。

PowerShell
# お子様（元のユーザー）の名前に戻す
git config --local user.name "お子様のGitHubユーザー名"

# お子様のメールアドレスに戻す
git config --local user.email "お子様のメールアドレス"
※ もし「このゲームは親御さんの作品として管理する」のであれば、この手順2は不要です。
